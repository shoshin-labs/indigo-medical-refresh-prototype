# Indigo Medical refresh prototype

A presentation-ready static prototype exploring how Indigo Medical's website could be refreshed to make patient journeys clearer and more confidence-inspiring.

## Included
- `review.html` — sharp presentation-ready review for Dr Rogers
- `index.html` — refreshed homepage concept
- `appointments.html` — clearer appointments journey
- `patient-access.html` — simplified online access onboarding
- `prescriptions.html` — prescription journey concept
- `register.html` — registration journey concept
- `services.html` — fuller services overview
- `surgeries.html` — surgery information and location concept
- `team.html` — medical team concept
- `contact.html` — clearer contact and form routing
- `docs/review-for-dr-rogers.md` — markdown review note
- `docs/prioritised-improvement-list.md` — must/should/could list

## Run locally
```bash
cd /var/lib/hermes-agent/indigo-medical-refresh-prototype
python3 -m http.server 4173
```

Then open `http://localhost:4173/`.

## Notes
- This is an exploratory concept, not an official Indigo Medical website.
- Publicly visible Indigo Medical information has been reorganised here for review and prototype purposes.
- No build step is required.
