# Prioritised improvement list

## Must

### 1. Clarify the appointment journey
Create a single clear appointment decision area that answers four questions immediately:
- I need a routine appointment
- I need an urgent same-day appointment
- I want to book online
- I need out-of-hours help

### 2. Improve the online booking flow
Replace or simplify the current registration journey for Patient Access. If the back-office process cannot change yet, present it as a short, visual, step-by-step flow with clear expectations.

### 3. Make contact routes explicit
If patients are told they can email, show the email address clearly. If not, remove that wording. Separate general enquiries from urgent clinical issues.

### 4. Improve form usability and accessibility
Use visible labels, mark required fields clearly, improve validation messaging, and explain what the contact form should and should not be used for.

## Should

### 5. Rebalance homepage calls to action
Make “Book an appointment” the primary action, with prescriptions, registration, Patient Access, and billing as secondary routes.

### 6. Add clearer reassurance around urgent care
Surface urgent same-day and out-of-hours guidance nearer the top of the site, so patients do not need to search for it.

### 7. Tighten content and copy
Reduce long paragraphs, convert key information into scannable cards or bullet points, and correct small spacing / spelling issues.

## Could

### 8. Introduce response-time expectations
For contact forms and admin queries, add guidance such as expected response windows and when to call instead.

### 9. Use more patient-task-led navigation
Frame the site around what patients want to do today, rather than only around practice information.

### 10. Refresh visual hierarchy
A lighter, cleaner layout with stronger section spacing and clearer button hierarchy would make the experience feel more modern without losing trust.
